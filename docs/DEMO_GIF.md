# Demo GIF（占位）

沟通维护者时，建议在仓库根目录或 `docs/` 下提供 **30 秒以内** 的录屏或 GIF：

1. 打开文本编辑或终端，切换到本输入法。
2. 输入一段拼音，待 AI 候选出现在第 2 位（或你配置的位置）。
3. 选中并上屏。

**建议文件名**：`docs/demo.gif`  

在 [README.md](../README.md) 的 Quick start 中取消注释：

```markdown
![Demo](docs/demo.gif)
```

**工具**：macOS `Cmd+Shift+5` 录屏 → `ffmpeg` / [gifski](https://gif.ski) 转 GIF；注意控制分辨率与帧数以控制体积。
